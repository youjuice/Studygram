## 💡 기획 의도

## 📎 서비스 소개
1. 풀어야 할 문제집만을 모아 **나만의 스터디 만들기**
2. 사용할 언어와 문제집 번호를 입력하면 **언어별 진도 현황 체크**
3. 백준 아이디를 통해 **백준 연동 회원가입**
